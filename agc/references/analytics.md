# AGC 分析服务

> **适用版本**：HarmonyOS 5 及以上
> **所属平台**：AppGallery Connect（AGC）

---

## 概述

AGC 分析服务为开发者和运营人员提供全面的数据分析能力，涵盖应用报表分析、元服务报表分析和行业风向标三大模块。通过分析报表数据，可深入了解用户行为、应用质量、销售与订阅情况，辅助产品决策与运营优化。

---

## 应用报表分析

### 概述

应用报表分析面向 HarmonyOS 应用，提供关键指标监控、使用情况分析、销售与订阅分析、质量分析等能力，帮助开发者全面掌握应用的运营状况。

### 核心能力

| 能力 | 说明 |
|------|------|
| 查看应用关键指标 | 实时监控应用的活跃用户、新增用户、留存率等核心指标 |
| 分析应用使用情况 | 了解用户在应用内的行为路径、页面访问、功能使用频率等 |
| 分析应用销售和订阅情况 | 跟踪应用内购买、订阅收入、付费转化等商业数据 |
| 分析应用质量 | 监控崩溃率、ANR 率等质量指标，定位质量问题 |
| FAQ | 常见问题解答 |

### 关键指标

- 活跃用户数（DAU/MAU）
- 新增用户数
- 用户留存率
- 应用使用时长与频次
- 付费转化率
- 崩溃率与 ANR 率

## 元服务报表分析

### 概述

元服务报表分析面向 HarmonyOS 元服务，提供关键指标监控、用户与使用情况分析、履约情况分析等能力，帮助开发者了解元服务的运营表现。

### 核心能力

| 能力 | 说明 |
|------|------|
| 查看元服务关键指标 | 监控元服务的活跃用户、新增用户、使用频次等核心数据 |
| 分析用户和元服务使用情况 | 了解用户如何使用元服务，包括访问路径、功能使用等 |
| 分析元服务履约情况 | 跟踪元服务的服务交付质量与履约表现 |
| FAQ | 常见问题解答 |

### 关键指标

- 元服务活跃用户数
- 元服务新增用户数
- 用户使用频次与时长
- 服务卡片展示与点击数据
- 履约完成率
