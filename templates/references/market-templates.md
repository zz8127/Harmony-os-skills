# 生态市场模板（Market Templates）

> **适用版本**：HarmonyOS 6 / API 22（稳定）；API 23（稳定）
> **市场地址**：https://developer.huawei.com/consumer/cn/market/prod-list?origin=template&ha_source=yzc&ha_sourceId=89000071

## 市场模板说明

华为生态市场提供经过官方审核的 HarmonyOS 应用模板，开发者可直接基于模板修改使用。

**特点**：
- 经过华为审核，UI/UX 符合 HarmonyOS Design 规范
- 工程结构完整，可直接导入 DevEco Studio
- 支持按行业/场景筛选

## 如何使用市场模板

1. **打开生态市场**：访问 https://developer.huawei.com/consumer/cn/market/prod-list?origin=template
2. **筛选模板**：选择「模板」Tab，按行业/场景筛选
3. **下载模板**：点击模板进入详情，下载 `.zip` 工程包
4. **导入 DevEco Studio**：`File → Open →` 选择下载的工程目录
5. **修改配置**：
   - 修改 `AppScope/app.json5` 中的 `bundleName`（全局唯一）
   - 配置签名（DevEco Studio → Project Structure → Signing）
6. **运行验证**：连接设备或启动模拟器确认功能正常

---

## 模板质量注意事项

- 下载的模板需自行配置 AGC 签名后方可运行
- 部分模板可能依赖特定的 SDK 版本（建议使用 API 22+）
- 模板中的第三方 SDK（如支付）需自行申请对应 AppID
- 模板仅供学习参考，直接上架需确保功能完整性和安全性
