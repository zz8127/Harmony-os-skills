# HarmonyOS 最佳实践

## 概述

HarmonyOS 应用开发最佳实践，基于大量真实开发场景提炼出的高频场景化案例，覆盖八大主题：架构设计、界面开发、功能开发、多设备开发、AI 使能、应用安全、应用质量、工程工具。旨在帮助开发者打造精致、易用、互联、智能、安全、流畅的 HarmonyOS 应用。

## 核心主题

### 架构设计
- 分层架构设计：确保系统具有良好的可扩展性、可维护性和可测试性
- 模块化设计：通过模块化拆分降低耦合度，提升开发效率

### 界面开发
基于 ArkUI 声明式 UI 开发框架，涵盖六大核心模块：
- 组件封装与复用
- 布局与弹窗
- 声明式语法
- 手势与导航
- 动画与转场
- 主题与样式

### 功能开发
应用框架、网络、媒体、图形、应用服务和 NDK 开发等重点领域场景化案例：
- ArkTS 高性能编程
- 应用间跳转
- 服务卡片
- 分段式拍照
- 音频焦点管理
- 水印添加

### 多设备开发
- 一次开发，多端部署（一多）：界面一多、功能一多、工程一多
- 全场景协同：多端协同、跨端迁移、跨端分享等自由流转能力

### AI 使能
- 基础语音服务
- 基础视觉服务
- 自然语言理解服务
- 意图框架：用户意图精准的智慧分发，实现主动预测推荐

### 应用安全
- 应用隐私保护
- 数据安全
- 安全编码
- 规范应用全生命周期的安全、隐私行为

### 应用质量
- 性能优化
- 功耗优化
- 稳定性保障
- 提高应用设计质量、运行效率和稳定性

### 工程工具
- 三方库移植
- DevEco Studio 高阶构建
- 定制个性化插件、产物

## npm 包

无独立 npm 包，最佳实践为指导性文档。
