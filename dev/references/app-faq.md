# 应用开发 FAQ

## 概述

HarmonyOS 应用开发常见问题解答（FAQ），涵盖应用框架开发各领域的常见问题与解决方案，帮助开发者快速定位和解决开发过程中遇到的疑难问题。

## 核心分类

### 程序包结构
- HAP / HSP / APP 包结构相关问题
- 模块配置与打包规则

### 程序框架
- Ability 生命周期管理
- UIAbility / ExtensionAbility 使用问题
- 页面路由与导航

### ArkTS 语言
- 方舟编程语言（ArkTS）常见问题
- ArkTS 线程模型和并发
- TypeScript 到 ArkTS 适配问题

### NDK 开发
- C/C++ 原生开发相关问题
- NDK 编译与链接

### UI 框架
- ArkUI 声明式开发范式问题
- 组件使用与自定义组件
- 布局与渲染

### 输入法框架
- 输入法接入与适配

### 本地数据和文件
- 数据持久化
- 文件访问与管理
- 数据库操作

### 无障碍和本地化
- 无障碍服务适配
- 多语言与国际化

### Web 框架
- Web 组件使用
- 网页加载与交互
- JSBridge 通信

### 多设备场景
- 手机端常见问题
- Pura X 等特殊设备适配问题

## npm 包

无独立 npm 包，FAQ 为指导性文档。
