# 字体规范

> **适用版本**：HarmonyOS 6 / HarmonyOS NEXT

## HarmonyOS Sans

HarmonyOS Sans 是华为为 HarmonyOS 设计的系统默认字体，基于汉仪字库与华为 UX 设计部联合开发。

### 字体特性

| 特性 | 说明 |
|------|------|
| 多语言支持 | 简繁中文、拉丁、西里尔、希腊、阿拉伯等 5 大书写系统，105+ 语言 |
| 可变字重 | 支持 Thin、Light、Regular、Bold、Black 等多种字重 |
| 动态字重 | 用户可选择字体粗细，系统根据观看距离/设备尺寸自动推荐最佳字重 |
| 时钟数字 | 冒号格式根据时间格式自动调整位置 |
| 深色模式适配 | 深色模式下使用更细字重，保持与浅色模式视觉粗细一致 |
| 等宽/变宽数字 | 支持变宽数字（文本段落阅读）和等宽数字（数据强调） |

### 字体易读性

- 中文笔画复杂，HarmonyOS Sans 对笔画连接处做出优化统一处理，效果干净利落
- 削弱了黑体的机械感，融入现代感结构处理和书写美感
- 比 Roboto 更显大更宽，与中文匹配度更高

## 字号层级（多设备）

HarmonyOS Sans 针对不同设备类型设计了对应的字号推荐值：

### 手机 / 折叠屏 / 平板

| 层级 | 推荐字号 | 使用场景 |
|------|---------|---------|
| 大标题 | 36-40 fp | 页面大标题 |
| 标题1 | 28-32 fp | 区块标题 |
| 标题2 | 24 fp | 卡片标题 |
| 标题3 | 20 fp | 次级标题 |
| 正文 | 16 fp | 主体内容 |
| 副文本 | 14 fp | 辅助说明 |
| 注释 | 12 fp | 标签、备注 |

### 智慧屏（大屏）

字号通常比手机大 1.5-2 倍，确保远距离可读性。

### 智能手表（小屏）

字号需更大或使用粗体，确保手腕距离可读。

---

## 折叠屏适配

| 要求 | 说明 |
|------|------|
| 展开态字号 | 单屏文本长度不超过 40 字，推荐 36 字左右 |
| 字号稳定性 | 展开态字号不应超过折叠态的 1.2 倍 |
| 可读性优先 | 优先保证展开态的可读性 |

---

## 字体使用原则

| 原则 | 说明 |
|------|------|
| 风格协调 | 字体风格与应用整体设计风格一致 |
| 层级清晰 | 通过字号/字重差异构建信息层级 |
| 避免硬编码 | 使用系统字体或 Token 变量，便于全局调整 |
| 国际化适配 | 考虑多语言字形差异，避免截断 |
