# 图标规范

> **适用版本**：HarmonyOS 6 / HarmonyOS NEXT

## 图标体系

HarmonyOS 提供两套图标系统：

| 图标类型 | 说明 |
|---------|------|
| HarmonyOS Symbol（新） | 矢量图标字库，1500+ 图标，支持分层颜色和动态效果 |
| 基础图标（传统） | PNG/SVG 格式，提供固定尺寸 |

---

## HarmonyOS Symbol（新图标体系）

HarmonyOS Symbol 是华为与汉仪于 **2024年6月** 联合发布的图标字库，是国内首次以字库形式封装的 Symbol 资源。

### 核心特性

| 特性 | 说明 |
|------|------|
| 矢量缩放 | 1500+ 矢量图标，任意分辨率/尺寸下保持清晰 |
| 分层结构 | 每个图层可独立设置颜色和灰度，支持精细调整 |
| 字重同步 | 无缝集成 HarmonyOS Sans，字体粗细变化时图标同步变化 |
| 动态效果 | 支持 7 种预设动画（消失、出现、弹跳等） |
| 操作绑定 | 动画可与点击、长按、输入、数据传输等用户操作绑定 |

### 使用方式（ArkTS）

```typescript
// 引用方式
SymbolGlyph($r('sys.symbol.ohos_folder_badge_plus'))
  .fontSize(96)
  .renderingStrategy(SymbolRenderingStrategy.SINGLE)
  .fontColor([Color.Black, Color.Green, Color.White])
```

### 渲染策略

| 策略 | 说明 |
|------|------|
| SINGLE | 单色模式（系统默认） |
| MULTIPLE | 多色模式（分层着色） |

### 动效类型

- 整体动效：图标整体作为一个图层播放动画
- 层级动效：图标的每个图层独立播放动画

## 基础图标规范

### 图标设计原则

| 原则 | 说明 |
|------|------|
| 可识别性 | 图标表意清晰，用户一眼可识别 |
| 风格一致 | 单个界面最多使用两种图标风格，线条粗细统一 |
| 颜色和谐 | 图标颜色与背景/文字有足够对比度 |
| 尺寸合适 | 图标尺寸与所在场景比例协调 |

### 热区与触控

| 要求 | 说明 |
|------|------|
| 图标热区 | 最小触控热区不低于 44×44 dp |
| 间距 | 图标与其他元素保持适当间距，避免误触 |

---

## 应用图标（App Icon）

应用图标需遵循华为应用市场上架规范：

| 要求 | 说明 |
|------|------|
| 尺寸 | 适配不同分辨率，提供多尺寸资源 |
| 格式 | PNG（推荐）、JPG |
| 风格 | 简洁清晰，与壁纸风格协调 |
| HMOS 角标 | 需遵循华为 HMOS 角标设计规范 |
| 品牌一致性 | 应用图标风格与品牌定位一致 |

---

## 主题图标

HarmonyOS 支持主题换肤，图标在主题中可自定义：

| 类型 | 说明 |
|------|------|
| 小主题 | 锁屏、桌面、图标范围换肤 |
| 大主题 | 锁屏、桌面、图标、联系人、信息、控制中心、拨号等全范围换肤 |
| 锁屏主题 | 仅锁屏范围换肤 |
| 图标主题 | 仅图标范围换肤 |
